Mathematics

I picked this project because it's one of the projects that shows growth of understanding, along with a better understanding of 3D geometry. This is the 3rd and final version of my 3D cube project. final_cube.py

2. A* Search Algorithm

This is an implementation of the A* search algorithm, the same one that is used by Google and other mapping applications to find the shortest path between two given points. It's a complex algorithm and although my implementation is far from perfect, it works well for something I made in a few hours. The project is also built using Python and the CMU Graphics library. It works by creating a grid, and then adding weights to each cell to find the path with the least "weight", also known as least time. It also uses heuristics to find the shortest path faster.

a_search.py

3. Slots

This is a slots game based off common 5x3 slot machines. It is a simple game using Python and the CMU Graphics library to simulate a slot machine. It's not entirely random and can be manipulated to win every time within the code, or if you're a casino, it can be manipulated to decrease odds. The most fun part of this project was determining the outcome of the spin, and testing if the "lines" were filled with the same symbol. I also added a free space that acts as a valid connector between any two symbols. It's incredibly easy to modify and even add new symbols to the game, being modular and expandable.

slots.py

4. Wave Function Collapse

This is my implementation of the Wave Function Collapse algorithm, a procedural generation algorithm that generates patterns based on a set of input tiles. The algorithm works by collapsing the wave function of each tile, which represents the possible states of the tile, until a valid pattern is generated. The project is built using Python and the CMU Graphics library, of course. However, this implementation suffers from some issues, such as bad performance and low optimization. I plan to revisit this project with a different Graphics library to both improve performance and optimization in another language, such as C++ or Javascript.

wave_function_collapse_2.py
